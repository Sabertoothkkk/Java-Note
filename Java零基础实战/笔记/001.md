### 代码规范

- 强制性代码规范，必须执行的

1、Java 程序的文件名与类名必须一致，若不一致，无法通过编译。

2、main 方法是程序的入口，方法的定义必须严格按照格式书写。

3、类是组织 Java 代码结构的，类中的方法是执行具体业务的。

- 非强制性代码规范，建议按照此方式编写代码

1、一行只写一条语句。

2、在 1 的基础上，还要注意代码缩进。
